IoT – Sensoren Uitlezen met Microcontroller

Voor dit IoT-project heb ik sensoren aangesloten op een microcontroller en de meetwaarden uitgelezen en doorgestuurd naar een dashboard.

Doel van het project

  • Sensoren aansluiten en correct bedraden.
  • Code schrijven om metingen uit te lezen.
  • Data op een overzichtelijke manier presenteren.

Gebruikte technologieën

  • • Microcontroller (bijv. Arduino of ESP32)
  • • Sensoren (bijv. temperatuur, licht, afstand)
  • • Seriële communicatie of eenvoudige webinterface

Wat heb ik geleerd?

Dit project heeft mij geholpen om mijn praktische vaardigheden verder uit te bouwen. Ik leerde niet alleen de technologie kennen, maar ook het belang van documentatie, plannen en gestructureerd werken.
